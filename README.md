# - Steps to run: 

Install Node and the requirements on Windows:
 1. winget install OpenJS.NodeJS
 2. npm install d3@3 jquery select2 (restart terminal if path is not updated)
 3. npm install http-server

Running the damn thing: 
  1. CD to directory
  2. npm install
  3. http-server
